      事件结合双向数据绑定实现todolist 待办事项 删除和增加

      anders 2019-5-25 vue

      # 1.增加和删除

      1.我们通过 v-on:click 或者 @click 来绑定一个事件 2.增加：对数组用 push()方法进行增加 3.删除：对数组用 splice()方法进行删除

      #数组：
todo: "",
list: [
  {
    title: "学习vue",
    checked: false
  },
  {
    title: "学习php",
    checked: true
  },
]
#增加
<input v-model="todo">
<button @click="doAdd()">+增加</button>
doAdd() {
  this.list.push({
    title:this.todo,
    checked:false
    }

#删除
<button @click="removeData(key)">删除</button>
removeData(key) {
  this.list.splice(key, 1);
}

      # 2.循环渲染和判断

      1.我们用 v-for 把数组循环渲染出来 2.用 v-if 来判断是否需要显示出来 tip:新版vue中，v-for 和 v-if不能写在同一个标签

      <h3>未完成</h3>
<ul>
  <li v-for="(item,key) in list" :key="item">
    <div v-if="! item.checked">
      <input type="checkbox" v-model="item.checked">
      {{item.title}}-----
      <button @click="removeData(key)">删除</button>
    </div>
    <br>
  </li>
</ul>

<h3>已完成</h3>
<ul>
  <li v-for="(item,key) in list" :key="item">
    <div v-if="item.checked">
      <input type="checkbox" v-model="item.checked">
      {{item.title}}-----
      <button @click="removeData(key)">删除</button>
    </div>
    <br>
  </li>
</ul>

      对check框绑定一个 checked 属性来决定数组是否显示

      #实例代码

      <template>
  <div id="app">
    <input v-model="todo">
    <button @click="doAdd()">+增加</button>
    <br>
    <h3>未完成</h3>
    <ul>
      <li v-for="(item,key) in list" :key="item">
        <div v-if="! item.checked">
          <input type="checkbox" v-model="item.checked">
          {{item.title}}-----
          <button @click="removeData(key)">删除</button>
        </div>
        <br>
      </li>
    </ul>
    <hr>
    <h3>已完成</h3>
    <ul>
      <li v-for="(item,key) in list" :key="item">
        <div v-if="item.checked">
          <input type="checkbox" v-model="item.checked">
          {{item.title}}-----
          <button @click="removeData(key)">删除</button>
        </div>
        <br>
      </li>
    </ul>
  </div>
</template>

<script>
export default {
  name: "app",
  data() {
    return {
      todo: "",
      list: [
        {
          title: "学习vue",
          checked: false
        },
        {
          title: "学习php",
          checked: true
        },
      ]
    };
  },
  methods: {
    doAdd() {
      this.list.push({
        title:this.todo,
        checked:false
        }
        );
    },
    removeData(key) {
      this.list.splice(key, 1);
    }
  }
};
</script>

      在vue当中，有了动态绑定,循环渲染和判断渲染 让我们可以更加便捷，快速，优雅的编程。
