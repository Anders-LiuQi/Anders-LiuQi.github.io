vue 初探之Vue的安装

anders 2019-5-23 vue

介绍: 1.Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。 2.Vue 的创始人是尤雨溪大神，也是我这个博客vuePress的开发者。 向大神致敬。

# 安装Vue

Vue的安装方式有两种，一种是在官网直接下载vue.js文件，然后直接用 <script>导入，这个我们就不多做介绍了。我们主要介绍第二种，用npm导入。

# 1.直接引入

打开https://vuejs.org/js/vue.js下载源码，复制到项目文件夹，用<script>导入。

# 2.用npm导入

#注意：用npm导入的话必须得先安装好npm，或者安装node,node中都带有npm

$ npm install vue 

#如果想要使用 GitHub 上 Vue 最新的源码，你需要自己构建！

git clone https://github.com/vuejs/vue.git node_modules/vue
cd node_modules/vue

npm install

npm run build

##安装vue-cil

###1.用npm全局安装

#全局安装脚手架工具vue-cli：
npm install --global vue-cli

#初始化vue项目
#若没有安装webpack，则先安装webpack
npm install -g webpack
vue init webpack myVue

#进入到myVue目录下，使用npm install 安装package.json包中的依赖
cd myVue
npm install

#运行项目：
npm run dev

这个时候就可以打开vue页面了 在浏览器上输入：localhost:8080，将会出现vue初始页面。
